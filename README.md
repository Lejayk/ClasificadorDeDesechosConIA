# ClasificadorDeDesechosConIA
Este es un proyecto introductorio al mundo de la IA: Desarrollar un sistema que pueda detectar y clasificar diferentes tipos de residuos utilizando técnicas de inteligencia artificial (IA) y visión artificial. El sistema debe ser capaz de reconocer residuos comunes (plásticos, papel, vidrio, orgánicos, etc.) a través de imágenes.
